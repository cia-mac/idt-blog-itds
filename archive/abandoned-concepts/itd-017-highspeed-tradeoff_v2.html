A high-speed camera is not a faster shutter. It is a budget problem with four variables: frame rate, resolution, memory, and record time. Fix any three and the fourth is decided for you. Push frame rate up and record time collapses. Drop resolution and you buy it back. Memory is the envelope. Every configuration is a choice about which reality you want to capture.
