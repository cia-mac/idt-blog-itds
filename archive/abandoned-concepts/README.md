## Abandoned ITD concepts

Concepts that did not survive user review. Preserved for reference so the thinking is not lost.

- **ITD-017 High-speed tradeoff** (`itd-017-highspeed-tradeoff_v1/v2.html`) — three-slider fps/resolution/memory tradeoff with accumulating memory bar. Rejected: "this concept is not working."
- **ITD-018 Time magnification** (`itd-018-time-magnification_v1/v2.html`) — dual-scale timeline showing real-time vs viewer-time. Balloon pop (v1), electric arc (v2). Superseded by ITD-020's pure slow-motion pattern.
- **ITD-019 Frame filmstrip** (`itd-019-filmstrip_v1/v2.html`) — Muybridge-style contact sheet of captured frames. Rejected: "I don't want frames like that, doesn't do anything for me."

Pattern that replaced them: one canvas, one pivoting lever, one subject. See `generic/itd-020-slow-motion_v8.html`.
