A high-speed camera is not a faster shutter. It is a three-way budget between how fast you sample time, how many pixels you resolve in each sample, and how long you can keep recording before memory fills. The sensor, the readout electronics, and the RAM are the physical limits. Every configuration is a choice about which reality you want to capture. You cannot have all three.
